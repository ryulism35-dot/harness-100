### 일본인 관광객이 많이 찾는 부산 포인트

1. **남포동·자갈치시장** — 부산 최대 재래시장, 일본 여행 잡지 단골 소개 구역
2. **국제시장·BIFF 광장** — 한국 드라마 촬영지, K-스트리트 푸드 집결지
3. **용두산공원·부산타워** — 전망대에서 일본 느낌의 항구 전경 감상
4. **해운대 해수욕장** — 일본 NHK 등 매체에서 자주 소개하는 한국 대표 해수욕장
5. **광안대교 야경** — 요코하마 베이브리지와 비교되는 야경 명소
6. **기장 해산물 골목** — 신선한 대게·멸치 요리, 일본인 미식가 방문 다수
7. **롯데백화점 센텀시티** — 세계 최대 백화점 중 하나, 면세점 내 일본 브랜드 다수

### 일한 문화 차이 주의사항

| 항목 | 한국 | 일본과의 차이 |
|------|------|------------|
| 식당 호출 | 큰 소리로 "여기요!" 또는 테이블 벨 사용 | 일본처럼 조용히 기다리면 직원이 오지 않을 수 있음 |
| 물·반찬 | 대부분 무료 제공 | 일본과 달리 한국은 반찬 무한 리필 가능 |
| 팁 문화 | 팁 없음 | 한국에서 팁은 오히려 실례가 될 수 있음 |
| 식사 중 이동 | 자유롭게 이동 가능 | 테이블 공유·이동에 덜 엄격 |
| 계산 방식 | 식사 후 계산대에서 결제 | 테이블 계산이 아닌 카운터 결제가 일반적 |
| 지하철 음식 | 음식물 반입 가능하나 자제 | 일본처럼 음식 반입 절대 금지는 아님 |
| 에스컬레이터 | 부산은 오른쪽 서기 (서울과 반대로 혼재) | 지역마다 다르니 현지 흐름 따라가기 |
| 쓰레기 처리 | 지정 봉투 필요, 길거리 쓰레기통 드묾 | 편의점·카페에서 배출하거나 숙소 이용 |
| 가격 흥정 | 재래시장에서 가능 | 백화점·편의점은 정찰제 |
| 신발 착용 | 일부 한식당 좌식 공간은 신발 벗음 | 미리 확인 후 입장 |

## 2. 교통 가이드

### 부산 지하철·버스 이용법

#### T-money 카드 (교통카드)

- **구입처:** 편의점(CU, GS25, 세븐일레븐), 지하철 역무실, 공항 내 안내소
- **충전:** 편의점, 지하철 충전기(현금)
- **기본요금 (2026년 기준 예상):** 지하철 1,450원, 버스 1,350원
- **환승 할인:** 지하철↔버스 30분 이내 환승 시 추가 요금 없음
- **아이 요금:** 만 6세 이하 무료, 만 6~12세 어린이 요금 적용
- **일본인 주의:** 일본 Suica·PASMO 카드는 부산 대중교통에서 사용 불가. 반드시 T-money 또는 현금 준비

#### 유모차 접근성

| 구간 | 접근성 | 비고 |
|------|--------|------|
| 부산 지하철 | 양호 | 주요 역 엘리베이터 완비 (일부 구역역 제외) |
| 해운대역 | 우수 | 엘리베이터·에스컬레이터 완비 |
| 남포역 | 우수 | 엘리베이터 이용 가능 |
| 시내버스 | 불편 | 저상버스 일부 운행, 유모차 접기 필요한 경우 많음 |
| 마을버스 | 불편 | 유모차 탑승 매우 어려움, 권장 안 함 |

> **팁:** 지하철 역 진입 시 "장애인·유모차 엘리베이터" 안내판 따라 이동. 출입구 번호 사전 확인 추천.

### 9명 이동 시 최적 교통수단

| 상황 | 추천 수단 | 이유 |
|------|----------|------|
| 공항/항구 → 숙소 | 승합택시 2대 or 11인승 렌터카 | 짐 많고 아이 피로 고려 |
| 숙소 → 롯데워터파크 (Day 2) | 11인승 렌터카 | 워터파크 짐 많음, 주차 가능 |
| 숙소 → 남포동 (Day 3) | 지하철 | 해운대역 → 남포역 직통 (2호선·1호선 환승), 약 40분 |
| 해운대 내 이동 | 도보 + 전동킥보드(성인) | 해변로 1km 이내 도보 가능 |
| 야간 귀가 | 카카오택시 | 늦은 시간 안전하고 편리 |
| Day 4 숙소 → 공항/항구 | 11인승 렌터카 or 승합택시 2대 | 짐 수거 후 이동 |

#### 택시 이용 팁

- **카카오택시** 앱으로 예약 (일본어 인터페이스 없으나 지도로 목적지 지정 가능)
- **대형택시(승합택시)** 선택 시 6~8명 탑승 가능
- **국제시장·자갈치 주변** 택시 잡기 어려울 수 있음 → 앱 예약 권장
- **모범택시(검은색):** 일반 택시보다 20~30% 비쌈, 의사소통 더 원활한 경우 많음

### 기장 롯데워터파크 이동 방법

#### 렌터카 이용 (강력 추천)

- **이동 시간:** 해운대 숙소 → 워터파크 약 30~40분
- **주차:** 워터파크 전용 주차장 (무료 또는 유료, 사전 확인)
- **11인승 카니발·스타리아 렌터카:** 9명 + 유모차 + 짐 탑재 가능
- **예약:** 롯데렌터카, SK렌터카, 제주렌터카 (사전 온라인 예약 필수)

#### 대중교통 이용 (참고용)

- **동해선 전철:** 센텀역 or 해운대역 → 기장역 하차 → 셔틀버스 또는 택시 이용
- **소요시간:** 약 50분~1시간 (환승 포함)
- **단점:** 유모차 + 대형 짐 + 9명 이동은 매우 불편. 성수기 혼잡 심함
- **결론:** 9명 단체, 유모차, 워터파크 짐 고려 시 렌터카 필수

- **부산밀면 원조집 (남포동 인근):** 부산식 밀면 전문, 아이 육수 따로 요청 가능

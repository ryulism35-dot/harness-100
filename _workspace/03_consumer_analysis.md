> **선행 인계**: industry-analyst (TAM 32조원, SAM 7.6조원, SOM 1,330~1,520억원) / competitor-analyst ("청정 기저전력 통합 솔루션" 포지션 공백 확인)
> **분석 목적**: CHP(500MW+) / SMR + 직접 PPA 공급 대상 산업단지 입주 기업의 구매 행동, 의사결정 구조, 지불 의향 가격(WTP) 분석

## 고객 세분화

### 세분화 기준 및 논리

산업단지 전력 PPA 구매자는 네 가지 축으로 세분화한다.

1. **RE100 이행 압박 수준** (글로벌 공급망 규제 노출도)
2. **전력·열 수요 규모** (CHP 경제성 결정 변수)
3. **EU CBAM 탄소비용 노출도** (직접 비용 회피 동기)
4. **에너지 비용 민감도** (전체 원가에서 에너지 비중)

### 단계 1: 인지 (Awareness)

| 항목 | 내용 |
|------|------|
| **주요 트리거** | ① RE100 이행 기한 도래 ② KEPCO 전기요금 인상 충격 ③ EU CBAM 적용 통보 ④ 경쟁사 PPA 체결 소식 |
| **정보 탐색 채널** | 산업부·에너지경제연구원 정책 발표, 에너지 컨퍼런스 (KIER, KAERI 주최), 업계 협회, 유사 업종 벤치마킹, 컨설팅사 브리핑 |
| **의사결정자** | ESG팀장·에너지팀장 (인식 → 내부 검토 제안) |
| **핵심 페인포인트** | "RE100 이행 수단이 재생에너지 인증서(REC) 외에 없는가?" "안정적 전력을 탄소 없이 공급받을 수 있는 사업자가 있는가?" |
| **기회** | 에너지 컨퍼런스 발표, 산단관리공단 협력, 업종별 협회 파트너십을 통한 선제적 브랜드 노출 |

### 단계 2: 검토 (Consideration)

| 항목 | 내용 |
|------|------|
| **검토 주체** | 에너지팀(기술 검토) + CFO/재무팀(비용 분석) + ESG팀(탄소 인증 적합성) + 법무팀(계약 리스크) |
| **평가 기준** | ① 공급 안정성 (정전 허용 기준, SLA) ② 전력 단가 및 장기 고정 여부 ③ 탄소 인증 유효성 (RE100 인정 여부) ④ 열 공급 가능 여부 및 단가 ⑤ 공급사 재무 안정성·트랙레코드 |
| **내부 병목** | 에너지팀 vs CFO 시각 충돌 (에너지팀: 기술·안정성 중시 / CFO: 단가·조기 매몰비용 중시) |
| **핵심 질문** | "기존 KEPCO 공급 대비 실질 절감액이 얼마인가?" "RE100 이사회 승인 요건을 충족하는가?" "계약 해지 시 패널티는?" |
| **기회** | 정량적 비용 절감 시뮬레이션 툴 제공, 유사 업종 레퍼런스 케이스 공유, 무료 에너지 진단 서비스 선제 제공 |

### 단계 3: 협상 (Negotiation)

| 항목 | 내용 |
|------|------|
| **협상 참여자** | 발주사 CFO·에너지본부장 + 공급사 사업개발팀·법무팀 + 외부 에너지 컨설턴트 |
| **주요 협상 변수** | 기준 단가, 단가 조정 방식(고정/변동/혼합), 계약 기간, SLA(가용률·공급 보증), 탄소 인증 책임, 열 공급 연계 조건, 지분 참여 여부 |
| **페인포인트** | LNG 연료비 연동 조항에 대한 구매자 저항 (가격 고정 선호) / 탄소 인증 유효성 불확실성 / 공급자 신용도 검증 |
| **협상 교착 지점** | 단가 조정 공식 이견, SLA 위약금 수준, 계약 만료 후 설비 처리 조건 |
| **기회** | "탄소 프리미엄 절감분으로 PPA 단가 상계" 논리 제시 / CBAM 비용 회피 BEP 계산서 제공 / 지분 참여(JV)로 이해관계 정렬 |

### 단계 4: 계약 (Contract)

| 항목 | 내용 |
|------|------|
| **최종 의사결정자** | CEO/이사회 승인 (A급·B급 대기업) 또는 CFO 전결 (C·D급) |
| **평균 소요 기간** | 의향서(LOI) 체결 후 본 계약까지 6~12개월 |

| **핵심 조항** | 공급보증율(>99.5%), 탄소 인증 이전 조항, 단가 조정 트리거, 불가항력 조항 |
| **기회** | 공급 개시 전 단계별 마일스톤(인허가 취득, 착공, 시운전)을 계약서에 명시하여 신뢰 제공 |

### 페르소나 1: "RE100 기한 압박" 대기업 — 글로벌 수출 제조업 유형

```
이름     : 김상현 전무 (삼성전자 DS부문 에너지인프라팀장 유형)
연령/직급 : 52세, 전무이사
소속     : 반도체 대기업 에너지인프라본부
담당     : 사업장 전력·유틸리티 조달, RE100 이행 계획 총괄
```

**배경 및 상황**

- 소속 기업은 2040년 RE100 이행을 이사회에서 선언했으며, 2026년 현재 전체 전력 사용량의 28%만 재생에너지로 충당
- 애플·구글·MS 등 주요 고객사가 공급망 Scope 3 탄소 보고를 강화하며, 2027년부터 탄소 중립 사업장 납품 기업 우선 선정 방침 통보
- 평택 반도체 팹 3기 가동으로 전력 수요가 연 2.3GW 추가 발생 예정이나 KEPCO 계통 증설 일정이 불투명
- 태양광 REC는 연간 수요의 5% 수준에 불과하며 간헐성으로 팹 운전 적합 불가

**핵심 니즈 (Jobs-to-be-Done)**

| 순위 | 니즈 | 현재 솔루션의 한계 |
|------|------|-----------------|
| 1 | 24/7 무탄소 전력 확보 | REC는 시간 불일치 (밤·흐린 날 미공급) |
| 2 | 전력 공급 안정성 (99.99% 가용률) | KEPCO 계통 가용률 99.95%, 자체 백업 비용 과다 |
| 3 | 10년 이상 고정 단가 (원가 확실성) | KEPCO 산업용 요금 매년 변동, 예산 계획 불가 |
| 4 | 글로벌 RE100/CFE 인증 유효성 | CHP는 RE100 미인정, SMR은 CFE 인정 논의 중 |
| 5 | 사내 이사회 승인을 위한 ESG 보고 자료 | 계약 구조 복잡성으로 ESG 공시 어려움 |

**의사결정 구조**

```
ESG위원회(사외이사) ← 보고
         ↑
  에너지팀장(김전무) ── 에너지 컨설팅사 활용 ── 공급사 RFP 발송
         ↑
  구매팀 + 법무팀 (계약 검토)
         ↑
  CFO 최종 승인 (비용 정당화 요구)
```

**WTP 분석**

| 항목 | 수치 |
|------|------|
| 현재 KEPCO 산업용 단가 | 135원/kWh |
| RE100 이행 미달 시 공급망 손실 추정 | 연 3,000~5,000억원 (계약 손실 + 브랜드 훼손) |
| 탄소 프리미엄 지불 의향 | KEPCO 대비 최대 20~25% 할증 수용 |
| 최대 WTP | 162~169원/kWh |
| 협상 타깃 단가 | 145~155원/kWh (PPA 프리미엄 7~15%) |

**핵심 메시지 (공급사가 전달해야 할 가치)**

> "저희가 제공하는 것은 단순한 전력이 아닙니다. 24시간 탄소 추적 가능한 기저전력으로 귀사의 RE100 이행 경로를 확실히 해드립니다. 애플·구글 공급망 심사를 통과할 수 있는 유일한 국내 CHP/SMR 직접 PPA 솔루션입니다."

### 페르소나 2: "CBAM 탄소비용 절감" 소재 기업 — EU 수출 대기업 유형

```
이름     : 박성민 상무 (POSCO/LG화학 에너지전략팀장 유형)
연령/직급 : 48세, 상무이사
소속     : 철강·석유화학 대기업 에너지전략팀
담당     : 에너지 조달, 탄소 배출권 관리, EU CBAM 대응
```

**배경 및 상황**

- EU CBAM 2026년 본격 부과 시작으로 유럽 수출 철강·화학 제품에 톤당 탄소 비용 부과
- 현재 사업장 전력 탄소강도 450gCO2/kWh (KEPCO 그리드 평균) 적용 시, 연간 CBAM 추정 부담액 약 800억원
- 저탄소 전력 조달 시 CBAM 탄소 집약도 기준치 하향 가능 → 수출 가격 경쟁력 직결
- 열 수요가 연간 3.2TWh에 달해 전기와 열을 함께 공급받을 경우 비용 절감 폭 극대화
- 자체 발전소(LNG 열병합) 노후화로 2029년 설비 교체 필요 — 대체 공급원 검토 중

**핵심 니즈 (Jobs-to-be-Done)**

| 순위 | 니즈 | 현재 솔루션의 한계 |
|------|------|-----------------|
| 1 | CBAM 탄소비용 절감 (전력 탄소강도 인하) | KEPCO 그리드 평균 탄소강도 450g/kWh — CBAM 부담 큼 |
| 2 | 전력 + 스팀 통합 공급 (비용 효율화) | 전력과 열을 각각 다른 공급자에서 조달하여 비효율 |
| 3 | 장기 에너지 비용 고정 (원가 안정성) | LNG 가격 변동으로 자체 발전 원가 예측 불가 |
| 4 | 기존 발전설비 교체 타이밍 최적화 | 조기 교체 시 매몰 비용 발생 우려 |
| 5 | EU CBAM 탄소 보고 데이터 자동화 | 수동 산정 → 오류 리스크, 감사 비용 |

**의사결정 구조**

```
CEO (EU 규제 대응 최우선 지시) → 에너지전략팀장(박상무)
                                          ↓
                          외부 에너지·탄소 컨설팅사 의뢰
                                          ↓
                     공급사 3~4곳 RFP 동시 발송 (경쟁 입찰)
                                          ↓
                      에너지팀 + 재무팀 공동 평가위원회
                                          ↓
                              CFO/부사장 최종 승인
```

**WTP 분석**

| 항목 | 수치 |
|------|------|
| 현재 KEPCO 단가 | 128원/kWh |
| CBAM 탄소비용 절감 효과 (저탄소 전력 전환 시) | 연 400~600억원 절감 가능 |
| 열 공급 통합 시 추가 절감 | 연 150~200억원 |

### 페르소나 3: "전기요금 절감 우선" 중견 제조기업 — 반월·시화 유형

```
이름     : 이철호 이사 (반월·시화 산단 중견 금속가공업체 관리이사 유형)
연령/직급 : 44세, 관리이사
소속     : 연 매출 500억~2,000억원 규모 중견 제조업체
담당     : 총무·구매·에너지 관리 겸직
```

**배경 및 상황**

- 전기요금이 제조 원가의 18~22%를 차지하며, 2023~2025년 KEPCO 요금 3차례 인상으로 영업이익률이 5%p 하락
- RE100 의무는 없으나 1차 협력사인 현대차·삼성전자가 공급망 탄소 보고를 강화하며 간접 압력 증가
- 에너지 관련 전담 인력 없이 관리이사가 겸직, 복잡한 계약 구조나 기술 검토 역량 부재
- 소규모(30~80MW) 수요로 개별 직접 PPA 진입 장벽 높음 — 집합 계약 또는 산단 관리공단 통한 간접 PPA 필요
- 단기 현금흐름 압박으로 초기 투자 최소화 선호

**핵심 니즈 (Jobs-to-be-Done)**

| 순위 | 니즈 | 현재 솔루션의 한계 |
|------|------|-----------------|
| 1 | 전기요금 즉각적 절감 (연 3~8%) | KEPCO 요금 인하 불가, 대안 없음 |
| 2 | 간단하고 투명한 계약 구조 | 직접 PPA 계약 복잡도로 진입 장벽 높음 |
| 3 | 초기 투자비 없는 에너지 전환 | 자가 발전 설치 비용 감당 불가 |
| 4 | 공급 안정성 유지 (생산 차질 방지) | 불안정한 신규 공급자에 대한 불신 |
| 5 | 협력사 ESG 보고 대응 자료 | 탄소 데이터 수집·보고 역량 없음 |

```

         ↓

         ↓

```

## 핵심 인사이트

### Insight 1: A급 고객의 WTP가 가장 높고, 지금이 계약 골든타임

삼성전자·SK하이닉스 등 반도체 대기업은 RE100 이행 기한이 2030~2040년으로 설정되어 있어, 2026~2028년이 장기 PPA 계약 체결의 황금 시기다. 이 시기를 놓치면 경쟁자(SK E&S, 재생에너지 사업자)에게 시장 선점당할 위험이 있다.

### Insight 2: CBAM이 B급 고객의 WTP를 매년 상승시킨다

EU CBAM 부과 규모는 2026년부터 매년 증가한다. 포항·여수·광양 철강·화학 기업의 청정 전력 WTP는 CBAM 적용 규모에 비례해 2028~2030년까지 연 3~5원/kWh씩 상승할 가능성이 높다. 계약에 CBAM 연동 단가 조정 조항을 포함하면 수익성이 지속 개선된다.

### Insight 3: D급 고객은 집합 PPA로만 수익성 확보 가능

반월·시화 산단 중소기업 개별 계약은 영업비용 대비 수익이 낮다. 산단관리공단 또는 업종 협회와 협력하여 50~100개사를 묶은 집합 PPA 계약으로 규모의 경제를 만들어야 한다. 이 전략은 산단 전체 공급구역 지정과 연계할 때 시너지가 극대화된다.

### Insight 4: 구매 여정에서 에너지팀 + CFO 이중 설득이 필수

에너지팀(기술 안정성, 공급 보증률 중심)과 CFO(비용 절감, 장기 고정 단가 중심)는 평가 기준이 다르다. 단일 제안서로 두 이해관계자를 동시에 설득하기 어려우므로, 맞춤형 자료를 이원화 제공해야 한다. 특히 CFO에게는 10년 현금흐름 절감 시뮬레이션을 제공하는 것이 계약 체결 속도를 높이는 핵심 요소다.

### Insight 5: 레퍼런스 효과로 후속 계약 가속

SK하이닉스나 LG화학 같은 A·B급 1개사와 계약 체결 후 동일 업종·산단 내 동반 효과가 발생한다. 반도체 산단에서 한 기업이 CHP PPA를 체결하면, 동일 산단 내 경쟁사가 "우리도 RE100 대응에 뒤처지면 안 된다"는 심리적 압박으로 신속하게 후속 계약을 체결하는 경향이 있다. 1순위 계약 체결 후 6~12개월 내 동일 산단 추가 계약 집중 공략 전략이 필요하다.

### Insight 6: 열 공급 통합이 전환 비용을 높여 고객 락인 효과 극대화

전력만 공급할 경우 고객은 다른 PPA 사업자로 전환이 비교적 용이하다. 그러나 스팀·열수를 함께 공급하면 열 배관 인프라 교체 비용이 발생하므로, 고객의 전환 비용이 급격히 증가한다. CHP의 열 공급 역량은 단순한 부가 수익이 아니라, 장기 고객 잠금(Lock-in) 전략의 핵심 수단이다.
